# ZTE F670L V.9 Unlocked Firmware

This repository contains the unlocked firmware for the ZTE F670L V.9 router.

## Overview

The ZTE F670L is a GPON ONT (Optical Network Terminal) router commonly used by internet service providers. This repository provides access to unlocked firmware that allows for enhanced configuration and customization options.

## Installation Instructions

### Prerequisites
- Syrotech GPON OLT access
- Administrative privileges on the OLT system
- Firmware file from this repository

### Installation Steps

1. **Upload Firmware to OLT**
   - Add the firmware file to your Syrotech GPON OLT system

2. **Configure ONU**
   - Cross-reference the ONU code in the OLT interface
   - Locate your specific ONT device

3. **Manual Upgrade Process**
   - Select "mONU upgrade" option
   - Choose "Manual upgrade" method
   - Select the uploaded firmware file
   - Initiate the update process

4. **Monitor Progress**
   - Keep monitoring the update progress bar
   - Do not interrupt the process until completion
   - Wait for confirmation of successful installation

## Important Notes

⚠️ **Warning**: Flashing firmware carries risks. Ensure you have:
- Proper backup of original firmware
- Stable power supply during the process
- Technical knowledge to recover from potential issues
